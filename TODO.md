# lsc-utils: modern ls-color utilities

lsc-utils is a collection of utilities in Rust for working with ls-color output.

## lsc: directory and file viewer with configurable decorations

First replicate the features and output of colorls:

- <https://github.com/athityakumar/colorls/>
- i have a config file at `~/.config/lsc-utils/colors.yaml` to load from
- general options will exist at `~/.config/lsc-utils/config.yaml`

Create unit and integration testsj, man pages, and other documentation.  

Add rust syntax linting and formatting.j

Create `just` file for building, linting and testing.

If environment variable DEBUG=1, print debug infornmation to stderr.

Setup versioning, commit and push.
